== Description ==

This plug-in helps you setup a custom message with cool graphics for all your visitors. This message can be a welcoming season's greetings or may be a news for the new attractions on your website. It's like putting up a placard and be assured to grab your visitor's attention.

Please visit [our support forum](http://www.netchillies.com/forums) to report bugs, support and request more features.

[Checkout our other plugins](http://www.netchillies.com/category/wordpress-plugin/)

== Installation ==
1. Extract the Plugin Folder from the downloaded ZIP file.
2. Upload cj-xmas-message folder to your /wp-content/plugins/ directory.
3. Activate the plugin from the "Plugins" page in your Dashboard.
4. Configure the plugin from Settings > CJ X-mas Message options page.

For trouble free execution, this plugin requires your theme to have wp_head() and wp_footer() functions.
